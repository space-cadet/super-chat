## Design Decisions
- **Text-based workflow** (not DB-native) — simpler for a new project, no SQLite setup needed
- **Monorepo structure** — core library + React UI components in one repo
- **TypeScript-first** — all exports typed, no runtime JS
- **Vercel AI SDK v6** as peer dependency — consumers bring their own version
