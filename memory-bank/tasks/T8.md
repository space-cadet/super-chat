## Details

React components for rendering tool calls, results, and multi-agent chat. Framework-specific layer — core library is framework-agnostic.

## Components

### Tool UI
- `PendingToolCard` — expandable card with Approve/Reject buttons
- `ToolResultCard` — expandable card showing success/error state
- `ToolCallBadge` — compact indicator in message bubble

### Message UI
- `MessageBubble` — user/assistant message with metadata
- `AgentIdentityBadge` — colored dot + name for multi-agent
- `CitationBlock` — arXiv paper citations
- `TypingIndicator` — per-agent typing status

### Chat UI
- `ChatInput` — textarea with @mention autocomplete
- `ChatMessages` — scrollable message list
- `SessionSidebar` — conversation history
- `ActionBar` — New Chat, Settings, Council toggle

### Settings UI
- `SettingsPanel` — provider profile management
- `ProfileCard` — individual profile display
- `ProfilePicker` — add/select provider

## Design Decisions
- Tailwind CSS for styling (consumer can override)
- Icons as SVG (no icon library dependency)
- Mobile-responsive by default
- Accessibility: ARIA labels, keyboard navigation
