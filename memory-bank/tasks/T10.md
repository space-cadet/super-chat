## Details

A runnable demo application that showcases all features of the library. Used for development, testing, and as a reference implementation.

## Features
- Chat UI with all components
- Demo tools (calculate, weather, arxiv search)
- Multi-agent simulation (2-3 mock agents)
- Settings panel with provider profiles
- Session management
- RAG demo (mock paper retrieval)

## Tech Stack
- Vite for dev server
- React 18
- Tailwind CSS
- LocalStorage persistence
- Demo adapters (no API keys needed)

## Files
- `demo/App.tsx` — Demo application
- `demo/main.tsx` — Entry point
- `demo/index.html` — HTML shell
- `demo/vite.config.ts` — Vite config

## Design Decisions
- Demo is separate from library code
- Mock data for all external APIs
- No real API keys required to run
- Can switch to real adapters via settings
