## Details

A runnable demo application and real-world test suite that verifies super-chat works with actual LLM API calls. Two forms: a React web UI and a CLI test runner.

## React Demo App

### Features
- API key input screen (stores in localStorage)
- Full ChatApp component with real streaming
- Tool calling with DemoToolAdapter (calculate, weather, arxiv)
- Session management
- Tailwind CSS styling

### Files
- `demo/index.html` — HTML shell
- `demo/main.tsx` — Entry point with Demo wrapper
- `demo/vite.config.ts` — Vite config with aliases

### Usage
```bash
pnpm demo        # Build + serve at http://localhost:3456
pnpm demo:build  # Build for production
```

## CLI Real-World Test

### Features
- Makes REAL API calls to OpenRouter
- Tests simple chat, tool calling, and multi-step execution
- Reports TTFT (time-to-first-token) and total duration
- Verifies streaming, tool calls, and tool results

### Files
- `scripts/test-real.mjs` — CLI test runner
- `scripts/test-chat-cli.mjs` — Mock CLI test (no API key needed)

### Usage
```bash
OPENROUTER_API_KEY=*** pnpm test:real
```

### Test Results (2026-06-20)

- ✅ Tool Use (calculate) — 12345 * 67890 = 838,102,050, TTFT: 2309ms, Total: 3659ms
