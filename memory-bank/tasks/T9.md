## Details

High-level API that wraps AgentLoop + adapters into a single, easy-to-use class. This is what arxivite (and other consumers) will interact with.

## Design

```typescript
class ChatEngine {
  constructor(options: ChatEngineOptions);
  
  // Send a message and get streaming events
  sendMessage(text: string, options?: SendOptions): AsyncIterable<StreamEvent>;
  
  // Load/save sessions
  loadSessions(): Promise<ChatSession[]>;
  saveSession(session?: ChatSession): Promise<void>;
  
  // Current session management
  createSession(title?: string): ChatSession;
  switchSession(sessionId: string): boolean;
  getActiveSession(): ChatSession | null;
  
  // Tool management
  registerTool(name: string, handler: ToolHandler): void;
  getAvailableTools(): ToolDefinition[];
  executeTool(call: ToolCall): Promise<ToolResult>;
  
  // Settings
  updateSettings(settings: Partial<ChatSettings>): void;
  getSettings(): ChatSettings;
  
  // Streaming control
  stopStreaming(): void;
  get isStreaming(): boolean;
}
```

## Notes
- Framework-agnostic (no React dependency)
- Handles abort/cancellation via AbortController
- Supports both streaming modes: tool-capable (AgentLoop) and text-only (direct adapter)
- Session management delegates to PersistenceAdapter
- Tool registration supports both adapter-provided and custom runtime tools
- Auto-saves messages after each turn (user + assistant)
