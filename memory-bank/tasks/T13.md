## Details

Helper functions to format tool results for LLM consumption. arxivite's tools return complex objects (papers, bookmarks, profiles) that need to be rendered as readable text before being fed back to the LLM.

## Design

```typescript
// Formatters for common result types
function formatPapers(papers: RetrievedPaper[]): string;
function formatBookmarks(bookmarks: Bookmark[]): string;
function formatProfile(profile: UserProfile): string;
function formatSearchResults(results: SearchResult[]): string;

// Generic formatter
function formatToolResult(toolName: string, result: ToolResult): string;
```
